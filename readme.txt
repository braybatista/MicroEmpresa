en mysql se debe crear la base de datos 'microempresa', las tablas las creara automaticamente hibernate.

se debe correr el REST API en el puerto 8082;

modificar el archivo properties del proyecto spring para conectar con MySQL
la primera vez se debe configurar agregando la linea
#spring.jpa.hibernate.ddl-auto=create
las proximas veces se debe comentar

en caso que el proyecto angular tenga un error con el exportador de datos ejecutar el siguiente comando
npm i --save-dev @types/file-saver
